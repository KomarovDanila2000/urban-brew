Urban Brew Premium CMS

Что внутри:
- premium-дизайн для кофейни
- адаптивный сайт для смартфонов
- верхняя автоматическая лента с фотографиями
- собственная админка для редактирования:
  - цен
  - товаров
  - акций
  - галереи
  - отзывов
  - контактов
  - hero-блока
  - фотографий через раздел Медиа

Как запустить:
1. Залей все файлы на PHP-хостинг.
2. Убедись, что папки content и uploads доступны на запись.
3. Открой сайт через index.php
4. Админка: /admin/login.php

Стандартные данные для входа:
логин: admin
пароль: Admin123!

Важно:
- после первого входа измени пароль в файле includes/config.php
- если хостинг не открывает index.php автоматически, создай .htaccess или настрой индексный файл в панели хостинга
- для замены фото:
  1) зайди в админку
  2) открой раздел Медиа
  3) загрузи изображения
  4) скопируй путь
  5) вставь путь в нужное поле на странице редактирования

Рекомендуемые фото:
- темные кадры кофе
- интерьер в мягком золотистом свете
- крупные планы напитков
- бариста, подача, десерты

Структура:
- index.php — фронтенд
- admin/ — админка
- content/content.json — хранилище контента
- uploads/ — загруженные изображения
- assets/ — стили, скрипты, плейсхолдеры
