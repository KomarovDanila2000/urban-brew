# Urban Brew — GitHub Pages Edition

Это статическая luxury-версия сайта, которая работает на GitHub Pages.

## Что умеет
- тёмный премиальный дизайн
- мобильная адаптация
- автоматическая верхняя лента с фото
- редактируемый контент через `content/site.json`
- страница `admin.html`, которая помогает редактировать контент и скачать новый `site.json`

## Важно
На GitHub Pages нет PHP и базы данных.
Поэтому настоящей серверной админки здесь быть не может.

Этот проект решает задачу так:
1. сайт работает полностью на GitHub Pages
2. ты меняешь тексты, цены, акции и ссылки через `content/site.json`
3. либо открываешь `admin.html`, меняешь поля и скачиваешь новый `site.json`
4. заменяешь файл в репозитории

## Как опубликовать
1. Залей все файлы в репозиторий GitHub.
2. Открой `Settings` → `Pages`.
3. Выбери `Deploy from a branch`.
4. Branch: `main`.
5. Folder: `/root`.
6. Нажми `Save`.

Ссылка будет вида:
`https://твой-username.github.io/urban-brew/`

## Как менять фото
Положи свои изображения в `assets/images/`.
Потом измени пути к ним в `content/site.json`.

## Как менять цены и меню
Открой `content/site.json` и отредактируй блок `menu`.

## Как пользоваться admin.html
1. Открой `admin.html` после публикации сайта.
2. Измени поля.
3. Нажми `Скачать site.json`.
4. Замени старый файл `content/site.json` новым в репозитории.
